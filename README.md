## ğŸ¨ Customization

### Colors
The color scheme can be customized in `tailwind.config.js`:
- Primary colors: Blue gradient theme
- Dark theme: Various shades of slate/gray

### Animations
Animation timings and effects can be modified in:
- `tailwind.config.js` for custom animations
- `src/index.css` for additional CSS animations
- Individual components for specific animation delays

All content is extracted directly from the resume and can be updated in the respective component files:
- `src/components/Hero.js` - Personal information and contact details
- `src/components/Education.js` - Educational background
- `src/components/Experience.js` - Professional experience
- `src/components/Skills.js` - Technical skills
- `src/components/Projects.js` - Projects and achievements
